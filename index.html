Day-3 Basic Statistical Function

통계 공부 방법

  • 빈도주의자 vs 베이지안주의자
  • 빈도주의자 : 숫자를 셈(평균)
  • 베이지안주의자 : 확률 기반

기본 통계 함수

  • (데이터세트$범주형 변수)
  • mean() : 평균
  • var() : 분산 (편차의 제곱/n)
  • sd() : 표준편차 (root(편차의 제곱/n)) → 비교를 할 때 의미가 생김

  • 평균값이 같아도 분산과 표준편차를 통해 전혀 다른 그래프임을 알 수 있다
  • quantile() : 사분위수 (전체 수를 4등분 한 수) → 이상치 제거할 때 기준으로 사용
  • IQR() : 1~3분위 범위 (3분위 - 1분위)
  • summary() : 요약
  • 1st Qu : 1분위 (최솟값부터 25퍼)
  • Median : 중간값 (전체의 중간에 해당하는 값)
  • 3rd Qu : 3분위 (최솟값부터 75퍼)
  • Descriptive Statistics : 기술 통계(통계적 추론)

빈도분석

  • table(데이터세트$범주형 변수)

빈도의 비율

  • prop.table(table(데이터세트$범주형 변수))

행과 열의 비율 형식 맞추기

소수점 아래 지정

  • round(숫자 , 자릿수)

데이터 가공

  • 데이터 전처리를 위한 도구
    • dplyr
      • 10GB 이내
      • data.table에 비해 배우기 쉽다
    • data.table
      • 50GB 이상
      • dplyr에 비해 배우기 어렵다
    • 처리속도 차이때문에 나눔
  • 라이브러리 설치

데이터 처리

  • glimpse() : 데이터 미리보기
  • 이름 바꾸기

빈도분석

  • count(데이터세트, 변수)
  • table(데이터세트$변수)

열을 추출

  • 객체 ← 데이터세트 %>% select()

행을 추출

  • 데이터세트 %<% slice(행1, 행2)
  • 조건에 맞는 행을 추출하기
    • 데이터세트 %>% filter(조건1, 조건2)

    • 여러 조건 모두 만족할때 : &

    • 여러 조건 중 하나라도 만족 할 때 : |

열을 추출 후 조건에 맞는 행을 추출

  • data2 = data3
    • data2처럼 처리하는 방법이 보기 좋고 많이 쓰인다.

파생변수 만들기

  • 데이터세트 ← 데이터세트 %>% mutate(새변수=변수만들기 연산)
  • 복수의 파생변수 만들기
  • total과 mean의 열이 생성된 것을 볼 수 있다.

집단별 통계량

  • group_by()와 summarise()는 세트
  • 복수 변수를 집단 분류해서 통계 구하기
  • 분류 집단별 데이터 빈도 구하기
  • 분류한 집단별 빈도와 비율 구하기

Day-2 R DataFrame

벡터

정수형 벡터

  • 복수의 숫자를 가진 벡터를 만들 때는 c()

문자형 벡터

  • 문자형 벡터를 만들때는 (“”)

범주형 벡터

  • factor()
  • 서열, 비서열은 유기적
  • 문자열과 숫자형이 만나면 문자열 우선
  • c()는 숫자형이라 연산O, 문자형과 factor는 연산x
  • 벡터의 유형을 알때 class()
  • 왜 WHY? 입력조건을 알기 위해
  • ordered= TRUE 를 정의하면 서열 범주열
  • 비서열 범주열은 factor(), 서열 범주열은 “ordered”

데이터 프레임(표만들기)

  • 각각의 함수는 같은 수의 집합을 가지고있어야한다.
  • 함수 = data.frame(a, b, c) a, b, c에 해당하는 열을 가진 frame을 만든다
  • 함수$내용 = 함수속에 있는 내용을 vector로 접근
  • 함수[행,열] = 행, 열에 속해있는 내용을 출력
  • 함수[a:b, c:d] = a행b행, c열d열에 속한 내용을 출력
  • 함수$내용[n] = 함수속에 있는 내용 중 n번째의 것을 vector로 접근
  • str(함수) = 함수 속 모든 vector의 datatape과 class를 한번에 접근

데이터 추가

  • 새로운 함수를 정의해 준 후 원래의 함수 + 추가된 함수로 만든다

데이터 제거

  • 함수[-n, -m] = n행 m열의 내용을 제거
  • 함수[-c(n,m), -c(a,b)] = n행과 m행, a열과 b열을 제거

list

  • list(a,b) = a함수와 b함수를 묶기

Data 내보내기, 불러오기

  • 현재 있는 dir확인
  • 내보내기(x = data.frame or matrix만 가능)
  • 불러오기
  • 주의사항 : 경로를 항상 주의하기
  • exam은 예시

엑셀파일 불러오기

  • 패키지 “readxl” 설치해야함
  • 명령어 : read_excel()
  • read_excel(“파일명”,sheet=n) : 파일명을 가진 엑셀에서 n번째 시트 불러오기

txt파일 불러오기

  • 명령어 : read.table()
  • tap, ; , “,”로 구분된 파일이 각각있다.
  • tap으로 구분된 파일 : read.table(“파일명”)
  • ;으로 구분된 파일 : read.table(“파일명”, sep = “;”)
  • ,으로 구분된 파일 : read.table(“파일명”, sep = “,”)

spss파일 불러오기

  • 패키지 “foreign” 설치해아함
  • 명령어 : read.spss()
  • list 형식으로 넘어와서 형변환을 해줘야한다.
  • 명령어 : as.data.frame()

실무패키지

  • 데이터가 많을땐 read.csv() -> fread()
  • fread()는 data.table 패키지에 있다.
