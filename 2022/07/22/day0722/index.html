Day-22 Oracla Basic

SQL 공부법

  • 메인쿼리를 파악한 후
  • 서브쿼리를 파악

서브쿼리

  • 인라인뷰

    • FROM절에 사용하는 서브쿼리
    • 세미조인과 비슷

  • 복잡한 쿼리를 작성해야 할때 분할을 한 후 작은단위부터 진행

계층형 쿼리

  • START WITH 조건문 & CONNECT BY 조건문
  • CONNECT BY PRIOR

계층형 쿼리 심화학습

  • 계층형 쿼리 정렬
    • ORDER SIBLINGS BY

    • 연산자
      • CONNECT BY ROOT
        • 최상위 로우를 반환하는 연산자
      • CONNECT_BY_ISLEAF
        • 최하위 로우면 1을, 아니면 0을 반환하는 연산자
      • SYS_CONNECT_BY_PATH(colm,char)
        • 계층형 쿼리에서만 사용
        • 루트노드에서 시작해 자신의 행까지 연결된 경로 정보를 반환
      • CONNECT_BY_ISCYCLE
        • 무한루프가 일어났을때 오류의 원인을 찾을 때 사용
        • NOCYCLE과 함께 사용

WITH절

  • 서브쿼리의 가독성 향상을 위해
  • 순환 서브 쿼리

    • SEARCH

      • ORDER SIBLINGS BY와 같은 기능
      • DEPTH FIRST BY : 형제로우보다 자식로우가 먼저 조회
      • BREADTH FIRST BY : 자식로우보다 형제로우가 먼저 조회

WINDOW 함수

  • 행이 삭제되지 않고 유지
  • Ranking, 누적 계산, 이동 평균 계산 시 주로 사용
  • 분석함수(매개변수) OVER (PARTITION B …)

분석함수

  • ROW_NUMBER()

    • ROWNUM과 비슷

  • RANK(), DENSE_RANK()

  • ENDSE_RANK()

    • 같은 순위가 나오면 건너뛰지 않고 매겨진다

  • CUME_DIST()

    • 상대적인 누적 분포도 값을 반환

  • PERCENT_RANK()

    • 해당 그룹 내의 백분위 순위를 반환

  • NTILE(숫자)

    • 파티션별로 명시된 값만큼 분할한 결과를 반환해준다

  • LAG

    • 선행로우의 값을 반환한다.

  • LEAD

    • 후행로우의 값을 반환한다.

WINDOW절

  • BETWEEN () AND ()
    • 절의 시작과 끝을 명시
  • UNBOUNDED PRECEDING
    • 첫번째 로우가 시작저짐
  • UNBOUNDED FOLLOWING
    • 마지막 로우가 끝지점
  • CURRENT ROW
    • 시작및 끝 지점이 현재 로우가 된다
