Day-4 Missing Values And Image Processing

Missing Values

  • 빈칸(모르는 값 0이 아닌 값)
  • 결측치는 처리방법이 매우 여러가지라서 알맞은 방법을 찾아야 한다.
  • 결측치가 지나치게 많으면 분석안한다

결측치 확인 함수

  • is.na(데이터세트) = IS this your? True / False

빈도 구하는 함수

  • table(데이터세트)

결측치 처리방법

결측치를 제외하고 분석

  • 연산함수에서 제외하는 함수
    • na.rm=T
  • 결측치가 있는 행 모두 제외하는함수
    • na.omit()
    • 가급적 쓰지 말것
  • 결측치가 없는 행만 추출
    • filter(!is.na())
  • 다른값으로 대체
    • imputatuion
      • A Solution to Missing Data: Imputation Using R
      • R 결측값(NA) 제거, 대체 방법

이상치

  • 데이터의 특정 값이 ‘이상’이 있다
  • case 1: 정해진 범주에서 벗어난 데이터 -> 9999
  • case 2: 숫자 / outlier /극단값
    • 평균을 내야하는데 극단적인 데이터들
  • trans의 갯수 구하기
  • ifelse()
    • 조건문

극단치 처리

숫자 데이터 boxplot()

  • boxplot()을 통해 극단치 유무 확인 가능
  • IQR : 3사분위 - 1사분위
  • 경계값 : IQR * 1.5
    • 상한값 : IQR + IQR * 1.5
    • 하한값 : IQR - IQR * 1.5

  • 극단치를 결측치 처리

Image Processing

데이터 시각화는 데이터 가공 이후에 진행

기본 시각화

  • ggplot() : 도화지
  • aes(x= , y=) : x축, y축 그리기기
  • 기본문법 : ggplot(데이터세트, aes(x= , y= )) + 그래프 종류

산점도 : x축 수치형(연속형)데이터, y축

수치형(연속형)데이터

막대그래프

  • 원자료에서 범주별 크기를 구해서 그리기

  • y값 지정

  • 범주별로 정해진 값을 토대로 그리기

히스토그램

  • 데이터의 분포가 어느구간에 몰려있는지 확인
  • coord_flip() : x축이 복잡할때 사용

ggplot()정교하게 그리기

  • 옵션명의 위치가 중요
  • geom_point(옵션=””)
    • 설정한 색상으로 전부 변환
  • ggplot(데이트세트, aes(x, y, ,옵션 = 컬럼명))
    • 컬럼별로 색상이 변환

산점도 그리기

###막대그래프에 2개 범주 내용 반영

  • dodge는 비교, fill은 비율
  • bar에만 있다
